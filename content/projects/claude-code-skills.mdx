## Overview

An open-source collection of specialized skills for Claude Code, Anthropic's AI coding assistant. The skills provide domain-specific knowledge and patterns for Python development, machine learning workflows, and quantitative finance.

## Skills Included

- **Python Programming** — Best practices, design patterns, type hints, testing
- **Machine Learning** — Model training, evaluation, deployment, MLOps
- **Quantitative Finance** — Backtesting, portfolio optimization, risk management
- **Software Engineering** — Architecture, CI/CD, clean code principles

## Design Philosophy

Each skill follows Anthropic's specification format with:
- Trigger patterns for automatic activation
- Domain-specific code templates and examples
- Anti-patterns and common pitfalls
- Integration with other skills for complex workflows
